# nicer-syntax If-Then-Else Statements

If statements are the main control flow.
There are three main forms: a single if, a single if-else, and a chained if-else.

```perl
# single if
if Condition, then
    do Something
done
# single if-else
if Condition1, then
    do Something
else, then
    do SomethingElse
done
# chained if-else
if Condition1, then
    do Something1
else if Condition2, then
    do Something2
else if Condition3, then
    do Something3
else, then
    do SomethingElse
done
```
